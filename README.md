# mailgun-sending-app

WebApp to send parameterised E-Mails.

## Dependencies

- npm (recommend [nvm](https://github.com/nvm-sh/nvm) to get it)

## Setup

```sh
# install dependencies
npm install

# build /dist folder
npm run build

# serve live app
npm run start

# format code
npm run format

# deploy to GitHub Pages (can take up to 10 minutes)
npm run deploy
```
